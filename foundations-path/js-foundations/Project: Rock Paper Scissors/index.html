In the last lesstion this game could only be able to play entirely from the console this version will allow the game to play from the Browser UI

Instructions to play the game

  • Gamer will have five rounds to play against the computer and in the end it will be shown who won the completion
  • Click on the below three buttons to select Rock, Paper or Scissors
  • Computer will randomly generate a choice and will be compared against the user choice
  • Final result will be shown on the screen
